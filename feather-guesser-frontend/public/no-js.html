How to Play

Getting Started

  1. Choose Your Mode: Select between Photo ID (identify birds from images) or Flip Mode (match descriptions to photos)
  2. Answer Questions: Each game consists of 10 carefully selected questions featuring different bird species
  3. Learn and Improve: Whether you answer correctly or incorrectly, you'll learn interesting facts about each bird
  4. Track Your Progress: See your score and try to improve with each game

Game Modes Explained

  • Photo ID Mode: Perfect for beginners and visual learners. You'll see a beautiful bird photograph and choose the correct species name from four options.
  • Flip Mode: More challenging! Read a description of a bird's characteristics, behavior, or habitat, then select the matching photograph.
